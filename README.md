============
Installation
============

Requires:
    
    - ip
    - iw
    - netctl
    - systemctl
    - noti

Update your PATH environment variable in your shell rc file with:
    
    $ export PATH="${PATH}":"/PATH/TO/PROGRAM/wifi"

Now the program is ready for use!
